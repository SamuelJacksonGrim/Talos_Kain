# diagrams/

The canonical diagram is the Mermaid flow chart embedded in
[`../../aamsfc.md`](../../aamsfc.md) — subgraphs 0–15, matching the walkthrough
sections. GitHub renders it inline.

This directory is for derived or exported views (rendered SVG/PNG, focused
sub-diagrams of a single organ, sequence diagrams for a specific flow) as they
get made. Keep the `aamsfc.md` chart as the source of truth; anything here is a
view of it, and if they ever disagree, the spec wins.

Intentionally empty for now.
